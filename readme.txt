Тестовое задание на вакансию web-разработчик
На сайте есть следующие страницы:
1) Главная страница - представляет собой список пользователей
2) Страница отправки заявки (/request) - представляет собой форму со следующими полями:
    - выбор получателя из списка пользователей
    - контактные данные отправителя
    - текст заявки
3) Страница пользователя (/user/([a-zA-Z0-9]+)). На ней данные пользователя + форма заявки без выбора получателя
4) Страница списка заявок /request-list) - представляет собой список всех заявок с постраничной навигацией. На одной странице 3 заявки.
5) Для решения задачи использовался микрофреймвор Slim PHP.

Верстка адаптивной.
